2 # Physically Dimensioned Quantities {#dimvals}
3 
4 Base class [dimval](\ref num::dimval) has public default constructor for
5 zero-valued quantity.
6 
7 ```cpp
8 #include <units.hpp>
9 dimval<-1,1,0,0,0> foo; // A speed with initial value of zero.
10 ```
11 
12 Each of dimval's template value parameters is an integer representing the
13 exponent of one of five fundamental units.
14 
15 1. second
16 2. meter
17 3. kilogram
18 4. coulomb
19 5. kelvin
20 
21 Aliases are available for convenience.
22 ```cpp
23 #include <units.hpp>
24 num::speed foo; // A speed with initial value of zero.
25 ```
26 
27 A wide array of units can be used for initialization and computation.
28 ```cpp
29 #include <units.hpp>
30 num::speed v = 2.3 * mi / hr; // Speed with initial value 2.3 miles/hour.
31 ```
32 
33 sqrt() and fabs() are supported.
34 
35 Member functions allow for integer power or root.
36 ```cpp
37 #include <units.hpp>
38 using namespace num;
39 length x = 2 * cm;
40 area a = pow<2>(x);
41 length y = 0.5 * sqrt(a);
42 volume v = pow<3>(x);
43 length z = 0.1 * root<3>(v);
44 ```
