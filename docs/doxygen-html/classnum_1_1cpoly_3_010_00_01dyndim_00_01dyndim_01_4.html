Detailed Description

template<>
class num::cpoly< 0, dyndim, dyndim >

Degree-zero specialization of cpoly with V = dyndim and C = dyndim.

cpoly<0,dyndim,dyndim> has but a single coefficient, the constant term.

Note that if either of V and C be dyndim, then the other must be, too.
