Detailed Description

template<unsigned D>
class num::cpoly< D, dyndim, dyndim >

Specialization of cpoly for V = dyndim and C = dyndim.

Note carefully that, in order to use dyndim, both the variable type and the term time must be dyndim. One must initialize the coefficients, in order from 0 to D, inclusive, before using any facility.
