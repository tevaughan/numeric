Detailed Description

template<typename A, typename F>
class num::dense_table< A, F >

A piecewise function that, in constant time, looks up the appropriate sub-function.

The argument \( a \) to the function is of type A.

For any \( a \), what is found in constant time are the corresponding sub-function and its sub-domain. The corresponding sub-function, stored in a table, is \( f_i \) of type F, and the center of the corresponding subdomain is \( a_i \), where \( i \in \{0, 1, \ldots, n-1\} \). The sub-argument \( a - a_i \) is passed to \( f_i \), and the value returned by \( f_i \) is returned from the lookup. In order for dense_table to return a value in constant time, every \( f_i \) must return a value in constant time.

In the simplest, fastest table, each \( f_i \) ignores its argument and returns a constant value. However, the infrastructure provided by dense_table allows for constant-time lookup of any kind of sub-function (for example, a cubic interpolant).

To enable constant-time lookup, an instance of dense_table stores

  • the argument-space location \( a_0 \) of the table's first record,
  • the fixed argument-space difference \( \Delta a \) between each record and the next, and
  • a list of \( n \) functions \( f_0, f_1, \ldots, f_{n-1} \), each of which (together with the computed \( a_i = a_0 + i \; \Delta a \)) makes up a record in the table.

Thus a regular grid is established across the value of the argument.

The lookup occurs by way of operator()(). When \( a > a_0 \) and \( a < a_{n-1} \), the offset \( i \) is found so that

\[ -\frac{\Delta a}{2} < a - a_i \leq +\frac{\Delta a}{2}. \]

What operator()() returns is \( f_i(a - a_i) \).

See sparse_table for a type that usually can approximate a given function so precisely as dense_table but with fewer records. The cost is logarithmic-time lookup for sparse_table.

template<typename A , typename F >
auto num::dense_table< A, F >::operator() ( A const &  a) const -> decltype(F()(a))
inline

For \( f_i \) and \( a_i \) associated with an argument \( a \) to the function, calculate \( f_i(a - a_i) \).

  • If \( a < a_0 \), then choose \( i = 0 \).
  • If \( a > a_{n-1} \), then choose \( i = n - 1 \).
  • Otherwise, choose \( i \) to be the integer nearest to \( \frac{a - a_0}{\Delta a} + 0.5 \).
Returns
\( f_i(a - a_i) \).
