Detailed Description

template<typename A, typename R>
class num::dense_table< A, R >

Simple, constant-time-lookup model of a function of a continuous variable.

dense_table stores a function's value at each point on a regular grid across the value of the function's argument. So each pair of subsequent table entries is separated by the same distance \(\Delta a\) in the space of the function's argument.

For log-time-lookup, see sparse_table.
