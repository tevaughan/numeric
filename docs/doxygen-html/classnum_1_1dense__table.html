Detailed Description

template<typename A, typename F>
class num::dense_table< A, F >

Constant-time-lookup model of a function of a continuous variable a of type A.

What is found in constant time is a tabulated function \( f_i \) of type F, where \( i \in \{0, 1, \ldots, n-1\} \). The tabulated function \( f_i \) is passed a transformed value of a, and the value returned by \( f_i \) is what is ultimately returned from the lookup. In the simplest, fastest table, each \( f_i \) ignores its argument and returns a constant value. However, the infrastructure provided by dense_table allows for constant-time lookup of any kind of interpolant, depending of the nature of F.

To enable constant-time lookup, an instance of dense_table stores

  • the first record's location \( a_0 \) in the space of the modeled function's argument,
  • the fixed difference \( \Delta a \) between each record and the next in argument space, and
  • a list of \( n \) functions \( \{ f_0, f_1, \ldots, f_{n-1} \} \), each of which, together with the argument value that can be computed as \( a_i = a_0 + i \; \Delta a \), defines a logical record in the table.

Thus a regular grid is established across the value of the modeled function's argument.

The lookup occurs by way of operator()(). When neither \( a < a_0 \) nor \( a > a_{n-1} \), the offset \( i \) is found so that

\[ -\frac{\Delta a}{2} < a - a_i \leq +\frac{\Delta a}{2}. \]

What operator()() returns is \( f_i(a - a_i) \).

See sparse_table for a type that usually can approximate a given function so precisely as dense_table but with fewer records. The cost is logarithmic-time lookup for sparse_table.

Calculate \( f_i(a - a_i) \) for tabulated function \( f_i \) associated with argument \( a \).

If \( a < a_0 \), then choose \( i = 0 \). If \( a > a_{n-1} \), then choose \( i = n - 1 \). Otherwise, choose \( i \) to be the integer nearest to

\[ \frac{a - a_0}{\Delta a} + 0.5. \]

Returns
\( f_i(a - a_i) \).
