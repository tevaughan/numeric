Detailed Description

template<typename A, typename F>
class num::dense_table< A, F >

Simple, constant-time-lookup model of a function of a continuous variable.

To enable constant-time lookup, dense_table contains a regular grid across the value of the function's argument. So each pair of subsequent table entries is separated by the same distance \( \Delta a \) in the space of the function's argument.

At the \( i^{\text{th}} \) entry, corresponding to the value \( a_i \) of the function's argument, an instance of dense_table stores a function object \( f_i \) of type F.

When the lookup occurs, by way of dense_table::operator(A const&), let the value of the passed-in argument be \( a \). When \( a \) is neither before the first table entry nor after the last, the index \( i \) is chosen so that

\[ -\frac{\Delta a}{2} < a - a_i \leq +\frac{\Delta a}{2}. \]

What is returned by dense_table::operator() is the value \( f_i(a - a_i) \).

For log-time-lookup, see sparse_table.

Template Parameters
AType of function's argument.
FType of each function \( f_i \), of which an instance of dense_table is a piece-wise construction.
