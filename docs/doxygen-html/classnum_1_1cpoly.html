Detailed Description

template<unsigned D, typename V = double, typename C = double>
class num::cpoly< D, V, C >

Model of a polynomial of a continuous variable.

The general template class cpoly is not fully compatible with statdim. The type C of each term may be a statdim, in which case type V must be dyndim. Similarly, if type C be dyndim, then type V must be dyndim.

However, the specialization cpoly<0,V,C> for the zeroth-degree polynomial (the constant) is fully compatible with statdim.

Class cpoly inherits from cfunc because cpoly is a model of a continuous function.
