Detailed Description

template<unsigned D, typename V = double, typename C = double>
class num::cpoly< D, V, C >

Model of a polynomial of a continuous variable.

Class cpoly inherits from cfunc because cpoly is a model of a continuous function.

Template Parameters
DDegree of polynomial.
VType of variable. By default, primitive type double.
CType of each term. By default, primitive type double.

§ cpoly

friend class cpoly
friend

Allow other type of cpoly to access private data.

This allows for multiplication of polynomials.

Ideally, only another cpoly with the same type of variable should be granted friendship, but there is no way to declare a partly specialized friend template.
