X  xu,

Extract a set of points from a space-delimited ASCII file.

Each line of the file must consist either

  • of only white space, optionally followed by the '#' character and a subsequent comment or
  • of white space followed by at least two space-delimited floating-point numbers (after which everything else on the line is ignored).
