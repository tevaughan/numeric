Detailed Description

template<typename A, typename F>
class num::sparse_table< A, F >

A piecewise function that, in logarithmic time, looks up the appropriate sub-function.

The argument \( a \) to the function is of type A.

For any \( a \), what is found in logarithmic time are the corresponding sub-function and its sub-domain. The corresponding sub-function, stored in a table, is \( f_i \) of type F, and the center of the corresponding subdomain is \( a_i \), where \( i \in \{0, 1, \ldots, n-1\} \). The sub-argument \( a - a_i \) is passed to \( f_i \), and the value returned by \( f_i \) is returned from the lookup.

In the simplest, fastest table, each \( f_i \) ignores its argument and returns a constant value. However, the infrastructure provided by sparse_table allows for logarithmic-time lookup of any kind of sub-function (for example, a cubic interpolant).

To enable logarithmic-time lookup, an instance of sparse_table stores a table of \( n \) triplets \( (a_0, {\Delta a}_0, f_0), (a_1, {\Delta a}_1, f_1), \ldots, (a_{n-1}, {\Delta a}_{n-1}, f_{n-1}) \), sorted so that \( a_0 < a_1 < \cdots < a_{n-1} \).

The lookup occurs by way of operator()(). When \( a > a_0 \) and \( a < a_{n-1} \), record \( i \) is found so that

\[ -\frac{{\Delta a}_i}{2} < a - a_i \leq +\frac{{\Delta a}_i}{2}. \]

What operator()() returns is \( f_i(a - a_i) \).

See dense_table for a type that usually can approximate a given function so precisely as sparse_table but with faster lookup. The cost is that dense_table requires more storage for the same precision with the same sub-function type F.

template<typename A , typename F >
auto num::sparse_table< A, F >::operator() ( A const &  a) const -> decltype(F()(a))
inline

For \( f_i \) and \( a_i \) associated with an argument \( a \) to the function, calculate \( f_i(a - a_i) \).

  • If \( a < a_0 \), then choose \( i = 0 \).
  • If \( a > a_{n-1} \), then choose \( i = n - 1 \).
  • Otherwise, choose \( i \) to be the integer nearest to \( \frac{a - a_0}{\Delta a} + 0.5 \).
Returns
\( f_i(a - a_i) \).
