Detailed Description

Namespace for C++-11 library enabling numerical computation.

The numeric library, whose symbols are encapsulated within the namespace num, defines facilities that include

  • compile-time dimensioned variables,
  • interpolation, and
  • numerical integration.

Construct a (sparse_table) piecewise-linear interpolant for a continuous function over the specified interval of its domain.

The initial number of evenly spaced samples should be sufficient to allow recursive subdivision to produce an interpolant with the desired fractional error tolerance. The numerical integral of the function is computed as a by-product and—if the user supply a pointer in the final argument—is then stored at the location indicated by the supplied pointer.

Template Parameters

iIf non-null, pointer to storage integral.
