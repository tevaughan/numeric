Detailed Description

template<char TI, char D, char M, char C, char TE>
class num::dimval< TI, D, M, C, TE >

Model of a statically dimensioned value.

For a dynamically dimensioned value, see dyndim.

Template Parameters
TIExponent of time
DExponent of distance.
MExponent of mass
CExponent of charge.
TEExponent of temperature.
