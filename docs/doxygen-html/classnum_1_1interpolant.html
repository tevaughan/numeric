Detailed Description

template<typename I, typename D>
class num::interpolant< I, D >

Linear interpolant.

This original C++ implementation is due entirely to Thomas E. Vaughan, who wrote it in 2016.

Class interpolant is designed with the idea that either of its template type parameters may be a double or else a dimval.

Template Parameters
IType of independent variable (x value).
DType of dependent variable (y value).

Multiply the present instance I1 by another interpolant I2 such that there is a point in the product interpolant I3 at every unique x coordinate across I1 and I2.

Each new y coordinate is the product of a y coordinate, in either I1 or I2, and the corresponding interpolated y coordinate in the other.
